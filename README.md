# I2C Master Controller

A synthesizable Verilog implementation of an I2C Master Controller with a complete verification environment.

## Features

- Standard I2C Master
- START and STOP generation
- ACK/NACK handling
- Multi-byte read/write
- Configurable clock divider
- Parameterized data width
- Self-checking testbench
- GTKWave support
